Publishing & SEO

- **Sitemap:** `sitemap.xml` is included at the repo root. After publishing, submit `https://freepirat.github.io/PiratDiment-Portfolio/sitemap.xml` in Google Search Console.
- **Robots:** `robots.txt` is included and references the sitemap.
- **Google Search Console:**
	- Add a new property using your site's URL (`https://freepirat.github.io/PiratDiment-Portfolio/`).
	- Verify ownership (HTML file or meta tag verification is convenient for GitHub Pages).
	- In Search Console > Sitemaps, paste `/sitemap.xml` and submit.
